# Image Editor Variations - Quick Reference

## How to Test NOW

1. Open your app
2. Go to any inventory item
3. Click on an image
4. Look for **V1 | V2 | V3** buttons next to "Photo Editor"
5. Click each to see the different layouts!

## The 3 Variations

### V1: Canva Classic
- **Look:** Sidebar on left with all tools
- **Best for:** Traditional workflow
- **Mobile:** Bottom toolbar

### V2: Minimalist Pro
- **Look:** Top bar + floating controls
- **Best for:** Maximum canvas space
- **Mobile:** FAB button (bottom-right) opens controls

### V3: Split Panel Pro
- **Look:** Before/After side-by-side
- **Best for:** Comparing changes
- **Mobile:** Stacked original/edited
