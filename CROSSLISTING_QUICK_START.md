# Crosslisting System - Quick Start Guide

Get started with the Vendoo-style crosslisting system in 5 minutes!

## Step 1: Connect Your Marketplaces

1. Navigate to **Marketplace Connect** (from sidebar or `/MarketplaceConnect`)
2. Click **"Connect"** for Facebook Marketplace
3. Complete OAuth flow
4. Repeat for eBay (if desired)

## Step 2: Create Your First Item

1. Go to **Crosslist Dashboard** (`/CrosslistDashboard`)
2. Click **"New Item"** button
3. Fill out the unified form:
   - **Title**: "Vintage Leather Jacket"
   - **Description**: Detailed description
   - **Price**: $50.00
   - **Condition**: Select condition
   - **Photos**: Upload 1-12 photos
   - **Shipping**: Configure shipping options
4. Click **"Save Item"**

## Step 3: Crosslist Your Item

1. In the **Crosslist Dashboard**, find your item
2. Click **"List"** button under Facebook column
3. Item is automatically listed on Facebook Marketplace
4. Status badge changes to "Active"
5. Repeat for eBay or other marketplaces

## Step 4: Bulk Operations (Optional)

1. Select multiple items using checkboxes
2. Click **"Bulk Actions"**
3. Choose action: **List**, **Delist**, or **Relist**
4. Select marketplaces
5. Click **"Execute"**
